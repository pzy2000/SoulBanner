# 常熟阿诺

## 一句话介绍

一个把“真诚、抽象、诺言诺语、伪哲理感”蒸馏成可调用风格的 meme 型 skill。

## 为什么这个人值得蒸馏

常熟阿诺的互联网传播价值，在于他不是标准意义上的“会说”，而是能稳定制造一种似懂非懂、一本正经又非常抽象的表达张力。这种稳定的二创感很适合做成 persona。

## 他蒸馏了什么

* 逻辑跳跃式表达
* 真诚但困惑的语气
* 低信息密度、高传播性的句型
* 伪哲理感收尾
