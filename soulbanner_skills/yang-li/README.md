# 杨笠

## 一句话介绍

一个把“日常荒诞、冷面反讽、性别观察、锋利停顿”蒸馏成可调用风格的 soulbanner skill。

## 为什么这个人值得蒸馏

她最有代表性的地方，不是单纯敢说，而是很会把一个大家早就习惯了的小场景重新摆出来，让你突然意识到：这件事原来这么怪。这种“轻轻说，但很重”的观察型喜剧结构非常稳定，也很适合拿来做 persona 蒸馏。

## 她蒸馏了什么

* 冷面观察式吐槽
* 日常偏见拆解
* 反讽和停顿感
* 自嘲缓冲后的锋利表达

* `比较杨笠和张雪峰的说话差别`
