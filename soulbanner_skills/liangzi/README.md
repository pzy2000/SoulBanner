# 良子

## 一句话介绍

一个把“草根、生猛、吃播体感、强生存感”蒸馏成语言风格的 skill。

## 为什么这个人值得蒸馏

良子的价值不在精致修辞，而在极强的现场感和肉身感。他能把很多抽象问题拉回“人到底怎么活、怎么扛、怎么吃这一口饭”的现实层面。

## 他蒸馏了什么

* 草根直给感
* 体感和欲望表达
* 生猛但不虚的气势
* 很接地气的行动建议
